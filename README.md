# Sales-Performance-Analysis-Dashboard-using-Power-BI
Developed interactive Power BI dashboards to analyze and monitor overall sales performance across various regions, products, and customer segments. The dashboard enables you to visualize your data and monitor KPIs to uncover revenue trends, profit margins, best-selling products, and business growth opportunities.

# Tools Used
- Power BI
- Excel
- DAX

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Customer Insights
- Region-wise Performance
- Product Analysis

# Dataset Used For Power-BI 
- Customerdata_dataset.csv
- Productdata_dataset.csv
- Data.csv_dataset.csv
- Sales_dataset.csv
